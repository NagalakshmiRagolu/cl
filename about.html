About This Project

This web application is built using Flask and multiple ML algorithms (KNN, Naive Bayes, Decision Tree, Random Forest, AdaBoost, Gradient Boosting, XGBoost, SVM).

The dataset used is the Iris Dataset, a classic dataset for ML beginners. It helps in classifying flower species based on their sepal and petal dimensions.
